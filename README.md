# Link Insight Tool

Understand what a link does before clicking it.

## What this tool does
- Previews link behavior without opening it
- Detects redirects
- Identifies downloads and login pages
- Highlights potential data collection forms

## What this tool does NOT do
- It does not block websites
- It does not label sites as scams
- It does not store or track user data

## Why this exists
Many users want to know what will happen when they click a link.
This tool provides transparency without judgment.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Serverless Functions (Vercel)
- No external APIs required

## Local Development
No setup needed.
Deploy directly on Vercel.

## Disclaimer
This tool provides a behavioral preview only.
Always be cautious when interacting with unknown links.
